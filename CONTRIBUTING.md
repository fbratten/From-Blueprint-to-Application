# Contributing

Thank you for your interest in this project!

## This Repository is Read-Only

This showcase repository is maintained by the author and **does not accept external contributions**.

### Why No Contributions?

This repository contains copyrighted book content and promotional materials. To maintain consistency and protect intellectual property, all content is managed directly by the author.

### Found an Issue?

If you've found a bug in one of the interactive demos or have feedback about the showcase:

1. **Do not** open a Pull Request
2. **Do not** fork and modify
3. Contact the author directly via [GitHub](https://github.com/fbratten)

### Want to Discuss the Book?

For questions about the book content, pre-release access, or bulk licensing:

- Visit the [pre-release page](https://subscribepage.io/from-blueprint-to-application)
- Contact the author via GitHub
